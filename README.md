# 🚀 Developer Portfolio

A fully responsive, interactive Developer Portfolio built with pure **HTML5, CSS3, JavaScript**, and **Three.js** — designed to showcase projects, skills, and personal information in a clean and modern layout.

## ✨ Features

### 🏠 **Hero / Landing Section**
- Animated introduction with name and role
- Interactive 3D background using Three.js
- Call-to-action buttons (View Projects / Contact Me)

### 👤 **About Section**
- Brief personal introduction
- Educational background
- Skills and interests overview

### 🖼️ **Projects Section**
- Cards showcasing featured projects
- Live demo and GitHub links for each project
- Project descriptions with technologies used

### 🛠️ **Skills Section**
- Visual display of technical skills
- Categorized by languages, tools, and frameworks
- Clean icon-based layout

### 📧 **Contact Section**
- Contact form for direct messaging
- Email and social media links
- Simple and clean form structure

## 📂 Folder Structure

```
Developer-Portfolio-main/
├── index.html          # 🏠 Main landing page (all sections)
├── style.css           # 🎨 Global stylesheet
├── script.js           # ⚙️  Main JavaScript logic
├── three-scene.js      # 🌐 Three.js 3D scene setup
├── README.md           # 📄 Project documentation
└── LICENSE             # 📝 MIT License
```

---

## ⚙️ How It Works

### Navigation Flow
```
index.html → Hero → About → Projects → Skills → Contact
```

### 3D Background Flow
```
three-scene.js → Scene setup → Camera → Animation loop → index.html
```

### Interactivity Flow
```
script.js → DOM events → Smooth scroll → Form handling → Animations
```
